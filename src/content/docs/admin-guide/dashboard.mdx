The admin dashboard is your home base at `/admin`. It shows all your clients and provides quick access to every part of the system.

## Client Dashboard

The main view lists all clients with their status, token, and quick-action links. From here you can:

- **Create a new client** - click the New Client button
- **Open a client hub** - click any client row to access research, generation, ideas, pages, and packages
- **View pitch site** - click the external link to see what the prospect sees

## Navigation

The admin sidebar is organized into sections:

### Sales Enablement

| Page                | Purpose                                       |
| ------------------- | --------------------------------------------- |
| **Active Pipeline** | View clients currently in progress            |
| **Partner Manager** | Add, edit, and organize partner organizations |

### Agents

| Page              | Purpose                                                                     |
| ----------------- | --------------------------------------------------------------------------- |
| **Single Client** | Single-URL interactive pipeline - enter one URL, generate everything        |
| **Multi Client**  | Batch pipeline - process multiple companies at once via the compute service |

### Admin

| Page               | Purpose                                                                  |
| ------------------ | ------------------------------------------------------------------------ |
| **AI Context**     | Global brand voice and positioning rules injected into all AI generation |
| **Cost Dashboard** | Track AI spend across all generation and agent jobs                      |
| **Job History**    | Browse completed and in-progress generation jobs                         |
| **Page Inventory** | Browse all generated pages across clients                                |

## Client Hub

When you click into a client, you see the client hub with tabs for:

- **Profile** - client details, token, status, partner selection
- **Research** - run Perplexity research, view/edit source of truth
- **Intelligence** - add notes and structured entries to enrich generation context
- **Generate** - create seed ideas, run full generation
- **Pages** - ideas, sales pages, partner pages, briefs, and packages
