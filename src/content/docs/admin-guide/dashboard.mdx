The admin dashboard is your home base at `/admin`. It shows all your clients and provides quick access to every part of the system.

## Client Dashboard

The main view lists all clients with their status, token, and quick-action links. From here you can:

- **Create a new client** - click the New Client button
- **Open a client hub** - click any client row to access research, generation, ideas, pages, and packages
- **View pitch site** - click the external link to see what the prospect sees

## Navigation

The admin sidebar is organized into sections:

### Sales Enablement
| Page | Purpose |
|------|---------|
| **Agent Mode** | Single-URL interactive pipeline - enter one URL, generate everything |
| **Active Pipeline** | View clients currently in progress |
| **Pages & Packages** | Browse all generated pages across clients |
| **Engagement Tracking** | Analytics on page views and interactions |
| **Partner Manager** | Add, edit, and organize partner organizations |

### Agents
| Page | Purpose |
|------|---------|
| **Pitch Agent** | Batch pipeline - process multiple companies at once via the compute service |
| **Proposal Agent** | Coming soon |
| **Content Agent** | Coming soon |
| **Outreach Agent** | Coming soon |

## Client Hub

When you click into a client, you see the client hub with sections for:

- **Overview** - client details, token, status
- **Research** - run Perplexity research, view/edit source of truth
- **Generate** - create seed ideas, run full generation
- **Ideas** - list of generated ideas with edit/archive controls
- **Pages** - sales pages, partner pages
- **Packages** - curated collections to share with prospects
