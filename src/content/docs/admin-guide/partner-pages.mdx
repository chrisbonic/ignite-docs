Partner co-sell pages showcase a joint value proposition between Ignite and a specific partner. They're designed for situations where you're co-selling with a partner organization.

## When to Use Partner Pages

- You're co-pitching with a partner (e.g., Webflow, Google Cloud)
- A partner's sales team wants a branded page to share with their prospect
- You want to highlight specific partner capabilities alongside Ignite's services

## Generation Flow

1. From the client hub, go to **Generate Partner Pages**
2. The system identifies **partner angles** - specific joint value propositions
   - Each angle focuses on how Ignite + the partner solve a problem together
   - Claude considers the partner's capabilities alongside the client's challenges
3. Review angles - approve, reject, or edit
4. Generate full partner pages from approved angles

## Partner Page Structure

Partner pages include everything a sales page has, plus partner-specific sections:

| Section                 | Content                                                 |
| ----------------------- | ------------------------------------------------------- |
| **Hero**                | Joint value headline                                    |
| **Proof**               | Combined social proof                                   |
| **Problem**             | Shared problem framing                                  |
| **Approach**            | Joint solution architecture                             |
| **Partner Focus**       | Which partner this page features                        |
| **Partner Credentials** | Certifications, badges, capabilities                    |
| **Case Studies**        | Joint success stories (HTML cards)                      |
| **Integration**         | How the partner's platform integrates with the solution |
| **CTA**                 | Joint call to action                                    |

## Partner Branding

Each partner page features:

- The partner's logo alongside Ignite's branding
- The partner's badge (e.g., "Webflow Enterprise Partner")
- A joint value proposition statement

## Positioning

Remember the positioning rules:

- **Ignite leads**, partners are "brought to the table"
- Never say "powered by" (this minimizes Ignite's role)
- Partners complement Ignite's innovation capabilities
