## Partner Fields

| Field             | Description                                                     |
| ----------------- | --------------------------------------------------------------- |
| **Name**          | Display name                                                    |
| **Description**   | What this partner brings (1-2 sentences)                        |
| **Badge**         | Label shown on pitch cards (e.g., "Webflow Enterprise Partner") |
| **Logo URL**      | Partner logo image                                              |
| **Website URL**   | Partner website                                                 |
| **Categories**    | Capability tags (see below)                                     |
| **Contact Name**  | Primary partner contact                                         |
| **Contact Email** | Contact email                                                   |
| **Active**        | Shows in the partner selection UI                               |
| **Featured**      | Auto-selected as default for new clients and Agent Mode         |
| **Sort Order**    | Display order in selection lists (lower = first)                |

## Adding a New Partner

1. Go to `/admin/partners`
2. Click **Add Partner**
3. Fill in name, description, badge, logo URL, and categories
4. Set **Active** to make it available in partner selection
5. Set **Featured** if it should be auto-selected for new clients
6. Set **Sort Order** for display priority

## How Partners Appear on Pitches

When you create a client, you select partners from the master list. The selection order determines prominence:

- **Position 1** - Most prominent partner card
- **Position 2-4** - Additional partner cards in order

Ignite Studio always appears as the first card (hardcoded), followed by the partners you selected.
