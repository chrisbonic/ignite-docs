Agent Mode is the fastest way to go from a company URL to a complete pitch package. Enter one URL, and the system runs the entire pipeline interactively.

## How It Works

1. Navigate to **Agent Mode** from the admin sidebar
2. Enter the company's website URL
3. The agent runs the full pipeline as a single streaming operation:
   - Company lookup (Perplexity)
   - Research generation
   - Seed idea creation
   - Full idea generation
   - Package assembly
   - CMS publishing
4. Progress streams back in real-time via SSE
5. When complete, you get a link to the finished package

## When to Use Agent Mode

- **Quick demos** - need a pitch package fast for a single company
- **Testing** - validate the pipeline end-to-end
- **Single prospects** - when you only have one company to process

## Agent Mode vs. Pitch Agent

| Feature     | Agent Mode                    | Pitch Agent                      |
| ----------- | ----------------------------- | -------------------------------- |
| Input       | Single URL                    | Multiple URLs                    |
| Backend     | Runs on admin (Webflow Cloud) | Runs on compute service (Vercel) |
| Interaction | Real-time streaming           | Prepare/review/launch workflow   |
| Review step | No seed review                | Review seeds before generating   |
| Best for    | Quick one-offs                | Batch processing                 |

<Aside type="note">
  Agent Mode runs directly on the admin server (Webflow Cloud). It works within Cloudflare Workers'
  timeout limits by streaming results via SSE with keepalive heartbeats. For larger pipelines or
  multiple companies, use the [Pitch Agent](/agents/pitch-agent/).
</Aside>

## Default Settings

Agent Mode uses sensible defaults:

- **Featured partners** are auto-selected
- **Audience lens** defaults to "Innovation Led Growth"
- All generated ideas are included in the default package
