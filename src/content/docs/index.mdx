## What is Ignite Studio?

Ignite Studio is an **On-Demand Innovation Acceleration Team** for enterprise prospects. The platform generates branded pitch packages - each client gets curated, unguessable URLs containing team pitches, idea pitch pages, sales pages, and partner co-sell pages.

<CardGrid stagger>
  <Card title="Create Clients" icon="add-document">

  <Card title="Generate Pitches" icon="rocket">
