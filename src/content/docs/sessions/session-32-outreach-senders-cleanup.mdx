## What shipped

### 1. Outreach sender presets

The outreach flow previously had a single hardcoded sender ("Chris Lutz / Head of Sales") embedded in [admin/src/pages/api/outreach/send.ts](https://github.com/chrisbonic/ignite-webflow-pitch-admin/blob/main/src/pages/api/outreach/send.ts). Now:

- **New file:** `admin/src/data/senders.json` — array of sender presets (Jesper first = default, Chris second). Each entry: `id, name, title, bio, email, linkedinUrl, initials, calendarUrl`.
- **Send form:** `/outreach/send` got a new "From" dropdown between Context and Model Selection. Shows `Name — Title` per option, defaults to first entry.
- **API:** `api/outreach/send.ts` now imports the JSON, exposes `resolveSender(senderId)`, falls back to first preset if missing/unknown.
- **Edit form:** `clients/[id]/pages/[pageId].astro` got a "Load preset" dropdown at the top of the Sender section. Picking one populates all 6 fields client-side (name, title, bio, email, linkedin, calendar). Save flow unchanged — still submits individual fields.

Existing pages keep their saved sender data. To change, open the edit form, pick a preset, save.
