The platform uses **Claude API** for content generation and **Perplexity Sonar** for research. Generated content is written directly to the Webflow CMS.

## Pipeline Flow

1. **Research** - Perplexity Sonar analyzes the prospect's website, news, and market data
2. **Quote Bank** - Claude + search finds 5-8 verified executive quotes with source citations
3. **Seed Ideas** - Claude proposes ideas with hero copy, category, challenge mapping, and matched quotes
4. **Review** - Admin approves/rejects/edits each seed before full generation
5. **Full Generation** - Claude generates all 40+ fields per approved seed (including visual HTML)
6. **CMS Write** - Content is written to the Webflow CMS via API
7. **Publish** - Items are published to make them live on the pitch site

## Two Pipeline Modes

| Mode | Entry Point | Backend | Use Case |
|------|-------------|---------|----------|
| **Agent Mode** | Admin `/agent.astro` | Admin SSE (direct) | Single URL, interactive |
| **Pitch Agent** | Admin `/agents/pitch.astro` | Compute service (Vercel) | Batch, multi-URL |

**Agent Mode** runs the full pipeline as a single SSE stream on the admin server. It does NOT use the compute service.

**Pitch Agent** uses the compute service for the same pipeline with a prepare/review/launch lifecycle and HTTP polling.

## Generation Workers (Compute Service)

The compute service runs specific generation tasks via `POST /api/generate/start` + `GET /api/generate/status` polling:

| Type | Purpose |
|------|---------|
| `seeds` | Generate seed ideas from research |
| `idea` | Generate single full idea |
| `batch-ideas` | Generate multiple ideas sequentially |
| `sales-angles` | Identify sales problem angles |
| `sales-page` | Generate sales pages from angles |
| `partner-angles` | Identify partner co-sell angles |
| `partner-page` | Generate partner pages from angles |
| `research` | Run Perplexity research via compute |

## AI Models Used

| Model | Purpose |
|-------|---------|
| **Claude** (Anthropic) | Content generation - ideas, sales pages, partner pages |
| **Perplexity Sonar** | Research - company analysis, quote discovery |

## Cost Tracking

Every generation job tracks:
- **Input tokens** and **output tokens** per AI call
- **Estimated cost** based on model pricing
- Aggregated cost view across all jobs

The cost dashboard at `/costs` aggregates data across all `gen:job:*` and `agent:job:*` entries in the KV store.

## Content Rules

All generated content must follow:
- **Hero h1**: MAX 3 lines, ~20 characters per line
- **Executive quotes**: Real, verified, sourced
- **Step 4**: Always "Your Team Presents" (hardcoded)
- **No em dashes**, no fake numbers, no SOWs/timelines/costs
- **Visual HTML**: Only safelisted Tailwind classes and `vis-*` helpers

## SSE Streaming Pattern

All AI endpoints on Webflow Cloud use SSE streaming with keepalive heartbeats every 2 seconds. This prevents Cloudflare Workers from killing idle connections.

The client-side code uses `getReader()` with line buffering to handle stream chunks that may be split mid-line by Cloudflare's TCP layer.
